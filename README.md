## 0. ПРОЧТИ ЭТО ПЕРВЫМ

**Ты думаешь, что играешь в игру про убийства.**
**Это самая дорогая ошибка твоего MMR.**

Ты играешь в **игру про конвертацию времени в карту**.

Каждая секунда → либо ресурс (голда, опыт, объект), либо пустота.
Убийства — **побочный продукт** правильного темпа, не его цель.

> Если после прочтения ты задашь себе один вопрос перед каждым действием —
> **«Что я делаю с этой секундой?»** — ты уже в топ-15% джанглеров своего ранга.

## 1. ТРИ ЛОЖНЫЕ МОДЕЛИ (и их замена)

Мозг не усваивает правила. Он усваивает **замену одной картины мира на другую**.

| ❌ Ты думаешь | ✅ Реальность |
|--------------|--------------|
| «Больше фрагов → быстрее ап» | Больше **объектов** → быстрее ап. Фраги без конвертации = ноль. |
| «Мне мешает команда» | Твой WR строго отражает **твои** решения. 40% матчей — твоя зона. |
| «Нужно выучить сложного пика» | Нужно выучить **3 вещи**: тайминги, маршрут, момент гэпа. |

## 2. КАРТА ИГРЫ: 15 МИНУТ, КОТОРЫЕ ВСЁ РЕШАЮТ

**Игра делится на 3 фазы. В каждой — одна главная задача.**

```
0:00 ── 2:00  ФАЗА ФАРМА       → Level 4 до 1:45. Без дедов. Без остановок.
2:00 ── 8:00  ФАЗА ЧЕРЕПАХИ    → Каждый Turtle = объект. Ганк EXP → Turtle → ганк.
8:00 ──12:00  ФАЗА ДАВЛЕНИЯ    → Lord как тул клоуза. Не награда — инструмент.
```

### Тайминги — наизусть, без сверки с гайдом

| Объект | Первый спавн | Респавн |
|--------|-------------|---------|
| 🟠 Orange Buff (Red) | ~1:30 | 75 сек |
| 🟣 Purple Buff (Blue) | ~1:30 | 75 сек |
| 🐢 Turtle | **2:00** | каждые **2 мин** |
| 👑 Lord | **≥8:00**¹ | каждые **3 мин** (до 18:00) → потом каждые **2 мин** |
| ☀️ Luminous Lord | **12:00** | — |

> ¹ Lord появляется ровно через **2 минуты после гибели последней Turtle**.
> Если Turtle жива в 8:00 и 5 секунд не получает урон → сама конвертируется в Lord.

**Цель:** Level 4 к **1:20–1:45** (про-стандарт). 2:00 — потолок, не цель.

## 3. МЕХАНИКА RETRI: ТОЛЬКО ТО, ЧТО МЕНЯЕТ ИСХОДЫ

**Retri — это не скилл убийства крипов. Это инструмент темпа и стила.**

### Базовые правила
- 5 стаков (крип / килл / асист) → апгрейд. Минионы не считаются.
- Первые **2 мин**: +15% снижения урона в своём джангле — используй для антиинвейда.
- Первые **5 мин**: Retri блокирует шаринг голды/опыта с союзниками рядом.
- **Маркс-джанглеры**: −50% урона Retri по крипам — учитывай в темпе фарма.

### Три апгрейда — точные механики

| Апгрейд | Эффект на героя | Кому |
|---------|-----------------|------|
| 🔵 **Ice** | Крадёт **72–100 ед. скорости** у цели на **4 сек** (враг теряет — ты получаешь) | Большинство ассасинов: Ling, Lancelot, Gusion, Julian, Karina |
| 🔴 **Flame** | Стилит **71–120 Phys ATK + Magic Power** у цели на **3 сек** (ты получаешь украденное) | Бёрст-пикоф: Hayabusa, Saber, Helcurt, Yi Sun-shin |
| 🩸 **Bloody** | Дрейнит HP цели несколько сек | Танк-джанглеры: Fredrinn, Arlott, Barats; Harley |

> **Стил через Retri:** Жди красную полоску HP объекта — только тогда жмёшь.
> Потраченный раньше Retri = подарок врагу.

### Какой бафф стартовый?
- **Purple Buff** → энерго/мана-зависимые (Fanny, Ling, Lancelot, Hayabusa)
- **Orange Buff** → авто-атакеры и бёрст-рейнджд (Yi Sun-shin, Helcurt)
- **Всё остальное** → смотри по сторону первого ганка, начни с ближнего баффа к цели

## 4. ДВИЖОК РЕШЕНИЙ: 4 МОМЕНТА, КОТОРЫЕ РЕШАЮТ ИГРУ

> Мозг работает быстрее, когда у него **дерево решений**, а не список правил.
> Сохрани эти 4 паттерна в оперативную память.

### Момент 1: Ганковать или фармить?

```
Turtle спавнится через > 60 сек?
├── ДА → ганк доступен? (враг оверэкстенд + ульт готов + ты в кустах)
│         ├── ДА → ганк, потом вернись в джангл
│         └── НЕТ → клири кемпы, не стой
└── НЕТ → стягивайся к Turtle. Ганкни EXP-лейн по пути.
```

### Момент 2: Контестировать Turtle или нет?

```
Вражеский пик сильнее в раннем файте?
├── ДА → НЕ контестируй. Стилни их бафф ИЛИ ганкни противоположный лейн.
└── НЕТ → заходи в яму за 15–20 сек до спавна. Turtle Lure если нужно.
```

### Момент 3: Брать Lord или нет?

```
Все 5 вражеских пиков на карте видны?
├── НЕТ → не берёшь. 1 невидимый пик = потенциальный стил + контест.
└── ДА → выиграли тимфайт последние 20 сек?
          ├── ДА → берёшь. Тиммейты давят другую линию.
          └── НЕТ → не берёшь. Ждёшь окна respawn таймеров.
```

### Момент 4: После тимфайта — что дальше?

```
Убили 2–3 врага → Turtle / ближайший тауэр (у тебя есть ~25–40 сек)
Вайпнули всех   → Lord / база (у тебя есть ~40–60 сек)
Зацепили и разошлись → джангл. НЕ зависай на лейне.
```

## 6. ВИЖН БЕЗ ВАРДОВ

> В MLBB нет покупных вардов. **Кусты = твоя единственная система вижна.**

- Перед Turtle / Lord → союзники чекают **оба входа в яму** кустами.
- Если куст не чекнут → ты не берёшь объект. Это правило без исключений.
- Трекинг через скорборд: скачок ~120 голды у врага = взял бафф. Считай 75 сек → его бафф открыт.

## 7. МЕНТАЛЬНЫЙ ПРОТОКОЛ

> Мышление — такой же скилл, как механика. Тренируется иначе.

**Три запрета в матче:**
- ❌ Не флеймишь — это передаёт инициативу эмоциям, не тебе
- ❌ Не коучишь — тилтит союзников, не помогает
- ❌ Не горишь на деде — каждый дед стоит 400–600 голды + пропущенный объект

**Три правила после матча:**
- ✅ Смотришь реплей → ищешь **одну** ошибку. Только одну.
- ✅ 2 поражения подряд → стоп. Тилт = экспоненциальные потери WR.
- ✅ Слитый матч ≠ потерянное время. Если нашёл паттерн — матч окупился.

> **Формула WR:** Ты контролируешь 40% матчей.
> При WR 55% → уверенный ап. При 60%+ → быстрый ап.
> Убери один тупой дед за матч — увидишь разницу за сезон.
